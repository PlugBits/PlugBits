よくある質問

無料トライアルはありますか？
はい。拡張機能の設定画面「Pro ライセンス」タブでメールアドレスを入力すると、 その場で14日間の Pro トライアルが始まります。クレジットカードの登録は不要です。
編集したデータは外部のサーバーを経由しますか？
いいえ。すべての処理はお使いのブラウザ内で行われ、データは kintone に直接保存されます。 PlugBits のサーバーを含め、外部のサーバーにレコードの内容が送信されることは一切ありません。
ライセンスキーはどのように受け取りますか？
購入完了後、ご登録のメールアドレスにライセンスキー（UUID形式）が送付されます。 Chrome 拡張のアイコン → 設定 → Pro ライセンスタブに入力してください。
解約はどこからできますか？
拡張機能の設定画面「Pro ライセンス」タブの「サブスク管理・領収書」ボタンから いつでも解約できます。解約後は翌月分の請求が発生しません。
無料版との違いは何ですか？
無料版ではサイドパネル（ショートカット・履歴・ウォッチリスト・Spreadsheet View の閲覧）と Quick New（レコード新規追加）が使えます。 Pro では Grid Edit による編集・保存・行追加と、Excel とのコピー＆ペーストが追加されます。
法人での購入・請求書払いは対応していますか？
現在は個人向けクレジットカード決済のみ対応しています。 法人でのご利用については support@plugbits.app までご相談ください。
複数アカウント・複数端末で使えますか？
1 ライセンスキーにつき 1 ブラウザプロファイルでのご利用を想定しています。 追加ライセンスが必要な場合はお問い合わせください。
