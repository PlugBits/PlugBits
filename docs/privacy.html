Privacy Policy

This privacy policy describes how the Chrome extension PlugBits Launcher for kintone handles user information.

Information Storage

The extension stores user configuration data using Chrome Extension Storage, including:

  • Pinned records
  • Favorite shortcuts
  • Recently accessed records
  • User interface preferences
  • Extension settings

The data is stored using:

  • chrome.storage.local
  • chrome.storage.sync

If chrome.storage.sync is used, settings may be synchronized between the user's Chrome browsers.

Network Communication

The extension communicates only with the user's kintone environment to retrieve data required for the extension features.

Supported domains include:

  • *.kintone.com
  • *.cybozu.com
  • *.kintone-dev.com

Personal Data Collection

The extension does not collect or transmit personal data to third-party servers.

The extension does not collect:

  • Personally identifiable information
  • Browsing history
  • Cookies
  • Tracking or analytics data

Data Sharing

User data is not sold, shared, or used for advertising purposes.

External Scripts

The extension does not load external scripts or third-party libraries.

Changes

This privacy policy may be updated when necessary.

Contact

If you have any questions about this policy, please contact us at:

support@plugbits.app

プライバシーポリシー

本プライバシーポリシーは、Chrome拡張機能 PlugBits Launcher for kintone におけるユーザー情報の取り扱いについて説明するものです。

保存される情報

本拡張は、以下の設定情報を Chrome 拡張ストレージに保存します。

  • ピン留めしたレコード情報
  • お気に入り（ショートカット）設定
  • 最近アクセスしたレコード
  • ユーザーインターフェース設定
  • 拡張機能の設定

保存には以下の Chrome 拡張ストレージが使用されます。

  • chrome.storage.local
  • chrome.storage.sync

chrome.storage.sync を利用する場合、Chrome の同期機能によりブラウザ間で設定が同期される場合があります。

通信について

本拡張は機能提供のため、ユーザーが利用している kintone 環境に対して API 通信を行います。

通信対象ドメイン:

  • *.kintone.com
  • *.cybozu.com
  • *.kintone-dev.com

個人情報の収集

本拡張は個人情報を第三者のサーバーへ収集・送信しません。

また、以下の情報を収集しません。

  • 個人を特定できる情報
  • ブラウザの閲覧履歴
  • Cookie
  • トラッキングや解析データ

第三者提供

ユーザーデータを第三者へ販売・共有することはありません。また、広告やマーケティング目的で利用することもありません。

外部スクリプト

本拡張は外部CDNや外部スクリプトを読み込みません。

変更

本ポリシーは必要に応じて更新される場合があります。

お問い合わせ

本ポリシーに関するお問い合わせは、以下までご連絡ください。

support@plugbits.app
