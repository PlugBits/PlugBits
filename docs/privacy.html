Privacy Policy

This privacy policy describes how the Chrome extension PlugBits Launcher for kintone handles user information.

Information Storage

The extension stores user configuration data using Chrome Extension Storage, including:

  • Pinned records
  • Favorite shortcuts
  • Recently accessed records
  • User interface preferences
  • Extension settings
  • Pro license key (if entered by the user)
  • License account information returned from verification (email address, expiry date) — stored locally only

The data is stored using:

  • chrome.storage.local
  • chrome.storage.sync

If chrome.storage.sync is used, settings may be synchronized between the user's Chrome browsers.

Network Communication

The extension communicates with the following services:

  • kintone environment (*.kintone.com, *.cybozu.com, *.kintone-dev.com) — to retrieve app metadata and records required for extension features.
  • api.plugbits.app — to verify the Pro license key entered by the user. Only the license key is transmitted; no kintone data or browsing history is sent.

No kintone record data is sent to external servers.

Personal Data Collection

The extension does not collect browsing history, cookies, or tracking data.

When a Pro license key is entered, it is transmitted to api.plugbits.app solely to verify license validity. The server returns the associated email address and expiry date, which are stored locally in Chrome storage to display account information within the extension. This data is not shared with third parties.

The extension does not collect:

  • Browsing history
  • Cookies
  • Tracking or analytics data
  • kintone record contents (these are processed locally and never transmitted externally)

Data Sharing

User data is not sold, shared, or used for advertising purposes.

External Scripts

The extension does not load external scripts or third-party libraries.

Changes

This privacy policy may be updated when necessary.

Contact

If you have any questions about this policy, please contact us at:

support@plugbits.app

プライバシーポリシー

本プライバシーポリシーは、Chrome拡張機能 PlugBits Launcher for kintone におけるユーザー情報の取り扱いについて説明するものです。

保存される情報

本拡張は、以下の設定情報を Chrome 拡張ストレージに保存します。

  • ピン留めしたレコード情報
  • お気に入り（ショートカット）設定
  • 最近アクセスしたレコード
  • ユーザーインターフェース設定
  • 拡張機能の設定
  • Proライセンスキー（ユーザーが入力した場合）
  • ライセンス認証で返却されるアカウント情報（メールアドレス・有効期限）— ローカルにのみ保存

保存には以下の Chrome 拡張ストレージが使用されます。

  • chrome.storage.local
  • chrome.storage.sync

chrome.storage.sync を利用する場合、Chrome の同期機能によりブラウザ間で設定が同期される場合があります。

通信について

本拡張は以下のサービスと通信を行います。

  • kintone 環境（*.kintone.com、*.cybozu.com、*.kintone-dev.com）— 機能提供に必要なアプリ情報・レコードの取得のため。
  • api.plugbits.app — ユーザーが入力した Pro ライセンスキーの認証のため。送信されるのはライセンスキーのみで、kintone のデータや閲覧履歴は送信されません。

kintone のレコードデータが外部サーバーに送信されることはありません。

個人情報の収集

本拡張は閲覧履歴・Cookie・トラッキングデータを収集しません。

Pro ライセンスキーが入力された場合、ライセンスの有効性確認のみを目的として api.plugbits.app へ送信されます。サーバーから返却されるメールアドレスと有効期限は、拡張機能内のアカウント表示のためにローカルの Chrome ストレージに保存されます。これらの情報は第三者へ提供されません。

本拡張が収集しない情報:

  • ブラウザの閲覧履歴
  • Cookie
  • トラッキングや解析データ
  • kintone のレコード内容（ローカルで処理され、外部に送信されることはありません）

第三者提供

ユーザーデータを第三者へ販売・共有することはありません。また、広告やマーケティング目的で利用することもありません。

外部スクリプト

本拡張は外部CDNや外部スクリプトを読み込みません。

変更

本ポリシーは必要に応じて更新される場合があります。
