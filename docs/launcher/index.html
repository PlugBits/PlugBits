Problem

kintoneは便利。
でも、ひとりで回すと、こまかい操作が一日中つきまとう。

kintoneをひとりで担当していると、
アプリを作る・レコードを入力する・一覧を直す、そのどれもがマウスでの細かい操作の積み重ねになります。
ひとつひとつは小さくても、一日の中で何度も繰り返すと、思考が途切れ、時間を奪われます。

設定画面までのクリックが多い

フォーム設定やフィールドコードの確認に、アプリ一覧から何度も階層をたどる必要があります。

入力は、要らない項目まで見せられる

全員向けに作られたフォームは項目が多く、自分が入力したい数個にたどり着くまでに目移りします。

一覧の修正が、1件ずつでしか進まない

複数レコードのちょっとした直しも、標準では1件ずつ開いて保存の繰り返しになります。

Solution

PlugBitsは、その操作をキーボードに置き換えます。

コマンドパレットで移動と設定を、クイック新規レコードで入力を、Grid Editで一括修正を。
マウスでの細かい操作を、キーボードの一撃に変えます。

キーボードで完結
操作の往復を削る

Free Features

まずは、毎日のこまかい操作を減らします。

PlugBits Launcherは、無料のChrome拡張として使えます。

コマンドパレット / Free

キーボードだけで、kintoneを操る。

Ctrl+/ で、どこからでも

ビューの切り替え、別アプリへの移動、レコード作成まで、名前を打つだけで一発実行。
マウスに持ち替える回数が減り、kintoneの移動が体感で変わります。

クイック新規レコード / Free

入力する項目は、自分で決める。

Shift+Alt+N で、すぐ入力

kintoneのフォームは全員のために作られているので、項目はどうしても増えていきます。でも入力する瞬間に必要なのは、そのうちの数個だけ。
クイック新規レコードは、自分に必要なフィールドだけを、自分の決めた並び順で表示する入力モーダルです。目移りせず、上から順に、最短で登録が終わります。
一覧画面から離れず、保存後もスクロール位置はそのまま。

スプレッドシート / Free

必要なときだけ、さっと開いて一覧を確認

補助ビューとして素早く開ける

スプレッドシートは、kintoneを置き換えるものではなく、必要なときにさっと開ける補助ビューです。
元の画面を離れすぎず、一覧の中で必要な情報をまとめて確認できます。

Side Panel

よく使う操作を1か所に集約

ショートカット

よく使うアプリをアイコンで固定。目的のアプリへすぐ移動できます。

ウォッチリスト

承認待ちなど特定ビューの件数を表示。今やるべき仕事の残数を確認できます。

レコードピン

気になるレコードはサイドパネルに固定。開きたいときにすぐ開けます。
