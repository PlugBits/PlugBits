## 2. 技術的進捗

### TypeScript 型チェック導入
- `npx tsc --noEmit` を正式な型チェック手段として採用
- Vite build が通っても tsc で落ちる問題を顕在化

### 現在の状態
- tsc 実行時に **22件の型エラー**を検出
- 主な内訳:
  - import に `.ts / .tsx` 拡張子が付いている問題（TS5097）
  - 型の widen（string 化）による union 不一致
  - CSSProperties の厳密化（wordBreak）
  - TableSummary / DataSource / TextElement 周りの型不整合

### 対応方針
- 仕様変更・UX変更は一切しない
- **型安全性のみを回復**
- CODEX に修正依頼プロンプトを作成済み
