PlugBits Reports 日本語フォント文字化け調査まとめ
2026-03-08
概要
PlugBits Reports の PDF 出力において、日本語テキストが文字化けする問題が発生した。
症状としては以下。
御見積書 → !
見積番号 → "#
発行日 → $%&
英数字は正常で、日本語のみ文字化けする。
結果として原因は
pdf-lib + fontkit + 日本語フォント + subset:true
の組み合わせで発生する フォントサブセット化バグだった。

原因
ログより
jp font head: OTTO
つまり
OpenType / CFF フォント
この条件
pdf-lib + fontkit + CFF font + subset:true
で 文字化けが発生する既知パターン。
解決方法
日本語フォントでは
subset:false
を使用する。
修正前
await pdfDoc.embedFont(jpBytes, { subset:true })
修正後
await pdfDoc.embedFont(jpBytes, { subset:false })
